# Daogreen Spec — чеклист v1.0 (production)

Обновлено под финальное ТЗ. Легенда: ✅ готово · 🟡 частично · ⬜ в плане

## Этап 1 — «реальный проект» (критично)

| Требование | Статус | Где |
|------------|--------|-----|
| Сервер + БД (не localStorage для данных) | ✅ | `backend/` SQLite-файл `data/daogreen.db` |
| PostgreSQL / .NET | ⬜ | Схема в SPEC.md; SQLite — рабочий стенд до миграции |
| Сохранение проектов, статусов, факт. цен, комментариев | ✅ | API `project_items` |
| Клиент по токену `/client/p/…` | ✅ | 24 байта `crypto.randomBytes` → base64url |
| Админ только для тебя (ключ API) | ✅ | `/login`, заголовок `X-Admin-Key` |
| Загрузка фото (URL / файл) | 🟡 | URL в материале; upload в `POST /upload-photo` |
| База материалов (поля SPEC) | 🟡 | см. §4 ниже |

> **localStorage** используется только для хранения ключа админа в браузере — не для проектов.
