# Фото деталей для базы материалов

Положите сюда изображения деталей стеллажа. Имена файлов:

- **По ID материала** (рекомендуется): `m001.jpg`, `m002.png`, `m003.webp`
- **По названию** (транслит): `труба_профильная_202015мм.jpg`

При старте API и по кнопке «Сканировать папку» в админке → **Фото** файлы автоматически привязываются к материалам.

Поддерживаемые форматы: jpg, jpeg, png, webp, gif.
